系统网络库

作用

通过 Python 代码模拟浏览器客户端发起网络请求获取响应数据，并且进行处理。

使用模块

  1. Python2 使用 urllib2 网络库
  2. Python3 使用 urllib 网络库

使用流程

  1. 导入模块
  2. 发起请求获取响应
    1. 定义请求地址
    2. 定义自定义请求头
    3. 定义请求对象
    4. 发送请求
  3. 处理响应内容

Python2

#!/usr/bin/python3
# -*- coding: utf-8 -*-
# 1. 导入模块
import urllib2
# 2. 发起请求获取响应
# 2.1 定义请求地址
url = "http://baidu.com"
# 2.2 定义自定义请求头
headers = {
    "User-Agent":"Mozilla/5.0 (Macintosh; Intel Mac OS X 10_14_0) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/71.0.3578.98 Safari/537.36"
}
# 2.3 定义请求对象
request = urllib2.Request(
    url=url,
    headers=headers
)
# 2.4 发送请求
response = urllib2.urlopen(request)

# 3. 处理响应
with open('01.html','wb') as f:
    f.write(response.read())

requests 模块使用

一、基本使用

使用方式
# 导入模块
import requests
# 定义请求地址
url = 'http://www.baidu.com'
# 发送 GET 请求获取响应
response = requests.get(url)
# 获取响应的 html 内容
html = response.text
代码讲解
  • response 常用属性
    • response.text 返回响应内容，响应内容为 str 类型
    • respones.content 返回响应内容,响应内容为 bytes 类型
    • response.status_code 返回响应状态码
    • response.request.headers 返回请求头
    • response.headers 返回响应头
    • response.cookies 返回响应的 RequestsCookieJar 对象
  • response.content 转换 str 类型
    # 获取字节数据
content = response.content
# 转换成字符串类型
html = content.decode('utf-8')
  • response.cookies 操作
    # 返回 RequestsCookieJar 对象
cookies = response.cookies
# RequestsCookieJar 转 dict
requests.utils.dict_from_cookiejar(cookies)
# dict 转 RequestsCookieJar
requests.utils.cookiejar_from_dict()
# 对cookie进行操作,把一个字典添加到cookiejar中
requests.utils.add_dict_to_cookiejar()

六、使用代理服务器

作用
  • 让服务器以为不是同一个客户端在请求
  • 防止我们的真实地址被泄露，防止被追究
使用代理的过程

代理分类
  • 透明代理(Transparent Proxy)：透明代理虽然可以直接“隐藏”你的IP地址，但是还是可以查到你是谁。
  • 匿名代理(Anonymous Proxy)：匿名代理比透明代理进步了一点：别人只能知道你用了代理，无法知道你是谁。
  • 混淆代理(Distorting Proxies)：与匿名代理相同，如果使用了混淆代理，别人还是能知道你在用代理，但是会得到一个假的IP地址，伪装的更逼真
  • 高匿代理(Elite proxy或High Anonymity Proxy)：可以看出来，高匿代理让别人根本无法发现你是在用代理，所以是最好的选择。

在使用的使用，毫无疑问使用高匿代理效果最好

从使用的协议：代理ip可以分为http代理，https代理，socket代理等，使用的时候需要根据抓取网站的协议来选择

使用方式
# 导入模块

# 定义请求地址

}

# 获取响应的 html 内容

发送请求时 proxies 参数设置代理

十、重试处理

使用方式
#!/usr/bin/python3
# -*- coding: utf-8 -*-
'''
可以使用第三方模块 retrying 模块
1. pip install retrying

'''
import requests
# 1. 导入模块
from retrying import retry

# 2. 使用装饰器进行重试设置
# stop_max_attempt_number 表示重试次数
@retry(stop_max_attempt_number=3)
def parse_url(url):
    print("访问url:",url)
    headers = {
        "User-Agent":"Mozilla/5.0 (Macintosh; Intel Mac OS X 10_14_0) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/70.0.3538.110 Safari/537.36"
    }
    proxies = {
        "http":"http://124.235.135.210:80"
    }
    # 设置超时参数
    response = requests.get(url,headers=headers,proxies=proxies,timeout=5)
    return response.text

if __name__ == '__main__':
    url = "http://www.baidu.com"
    try:
        html = parse_url(url)
        print(html)
    except Exception as e:
        # 把 url 记录到日志文件中，未来进行手动分析，然后对url进行重新请求
        print(e)
代码讲解

  • 安装 retrying 模块

    retrying 模块可以通过装饰器模式对某个函数进行监控，如果该函数引发异常就会触发重试操作

    pip install retrying

  • 对需要重试的函数进行装饰器设置

    通过 @retry(stop_max_attempt_number=重试次数) 参数设置重试次数

    # 1. 导入模块
from retrying import retry
# 2. 装饰器设置重试函数
@retry(stop_max_attempt_number=3)
def exec_func():
    pass

#!/usr/bin/python3
# -*- coding: utf-8 -*-
import requests
import json
# 定义请求url
url = "https://movie.douban.com/j/search_subjects"
# 定义请求头
headers = {
    "User-Agent":"Mozilla/5.0 (Macintosh; Intel Mac OS X 10_14_0) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/70.0.3538.110 Safari/537.36"
}
# 循环构建请求参数并且发送请求
for page_start in range(0,100,20):
    params = {
        "type":"movie",
        "tag":"热门",
        "sort":"recommend",
        "page_limit":"20",
        "page_start":page_start
    }
    response = requests.get(
        url=url,
        headers=headers,
        params=params
    )
    # 方式一:直接转换json方法
    # results = response.json()
    # 方式二: 手动转换
    # 获取字节串
    content = response.content
    # 转换成字符串
    string = content.decode('utf-8')
    # 把字符串转成python数据类型
    results = json.loads(string)
    # 解析结果
    for movie in results["subjects"]:
        print(movie["title"],movie["rate"])

    var b = document.cookie;

    for (var i = 0; i < c.length; i++) {

        if (a == d[0]) {
            return d[1]
        }
    }
    return ""
}

#!/usr/bin/python3
# -*- coding: utf-8 -*-
import requests
import js2py
from lxml import etree
import re
from bs4 import BeautifulSoup

context = js2py.EvalJs()

url = "http://datamining.comratings.com/exam"
headers = {
    "User-Agent":"Mozilla/5.0 (Macintosh; Intel Mac OS X 10_14_0) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/71.0.3578.98 Safari/537.36"
}

# 1. 从服务器获取 session
response = requests.get(url,headers=headers)
# 返回 RequestsCookieJar
cookies = response.cookies
cookies_dict = requests.utils.dict_from_cookiejar(cookies)
session = cookies_dict["session"]

# 2. 通过session生成 c1 和 c2
with open('data.js','r',encoding='utf-8') as f:
    context.execute(f.read())

c1 = context.get_c1(session)
c2 = context.get_c2(session)


# 3. 继续请求并且携带 session,c1,c2
#获取如果是 iframe 就提取出 iframe的 src 继续请求必须 session,c1,c2，
# 直到返回真正的数据

new_cookies = {
    "session":session,
    "c1":c1,
    "c2":c2
}

base_url = "http://datamining.comratings.com"
while True:
    print(url)
    response = requests.get(url, headers=headers, cookies=new_cookies)
    eroot = etree.HTML(response.text)
    src = eroot.xpath('//iframe/@src')
    if src is not None and len(src) > 0:
        src = src[0]
        # 继续请求
        url = base_url + src
    else:
        html = response.text

        with open('01-out.html','w',encoding='utf-8') as f:
            f.write(html)

        print("*"*50)
        # 使用正则表达式
        # 1. 把样式的内容替换进 html 内容中
        # 获取 style 字符串内容
        style_string = re.findall(r'<style>(.*?)</style>',html,re.DOTALL)[0]
        print(style_string)
        # 制作字典样式是key,value是{}里面内容
        styles = dict(re.findall(r'\.(.*?){(.*?)}',style_string))
        print(styles)

        body = re.findall(r'<body>(.*?)<body>',html,re.DOTALL)[0]
        # 把styles value替换进html中
        for key,value in styles.items():
            body = body.replace(key,value)

        # 2. 把 html 中的 display:none 从html内容剔除
        body = re.sub(r'<div (.*?)="display:none">(.*?)</div>',"",body)
        body = re.sub(r'<span (.*?)="display:none">(.*?)</span>', "", body)
        body = body.replace('\n','')

        body = body.replace('<br>','|')
        # 3. 把标签全部给删除只剩下标签内容
        soup = BeautifulSoup(body, 'lxml')

        print(soup.getText().split("|")[1:])
        break
