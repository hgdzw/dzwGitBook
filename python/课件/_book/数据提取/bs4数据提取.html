beautifulsoup4 介绍与安装

定义

和 lxml 一样，Beautiful Soup 也是一个HTML/XML的解析器，主要的功能也是如何解析和提取 HTML/XML 数据。

优点与缺点

  • 优点
    • 用来解析 HTML 比较简单，API非常人性化，支持CSS选择器、Python标准库中的HTML解析器，也支持 lxml 的 XML解析器。
  • 缺点
    • lxml 只会局部遍历，而Beautiful Soup 是基于HTML DOM的，会载入整个文档，解析整个DOM树，因此时间和内存开销都会大很多，所以性能要低于lxml。

官方文档

http://beautifulsoup.readthedocs.io/zh_CN/v4.4.0

安装

pip install beautifulsoup4

基本使用
  1. 导入模块
    from bs4 import BeautifulSoup
  2. 创建 beautifulsoup 对象 
    # 通过字符串创建, 第二个参数用于指定解析器
soup = BeautifulSoup(html,'lxml')
# 通过文件创建
soup = BeautifulSoup(open('文件路径'))
    注意：
    • 自动补全 html 和 body 标签
    • 自动补全缺省标签
  3. 打印输出
    print(soup.prettify())
  4. 获取元素标签元素
    # 默认返回第一个元素
print(soup.标签名)
  5. 使用 .contents 或 .children 获取子元素
    # 返回列表
soup.a.contents
# 返回迭代器
soup.a.children
  6. 获取元素内容
    soup.title.get_text()
  7. 获取元素属性值
    soup.a.get('href')
    注意：如果返回的是 class 属性返回的值是列表，如果返回的是其他属性返回的是单个值

find 与 find_all 元素查找

作用

用于查找定位元素

find 和 find_all 区别

  • find 返回符合条件的第一个元素
  • find_all 返回符合条件的所有元素列表

参数说明

find 与 find_all 提供参数都一样

  1. 通过标签或标签列表查找元素
    # 查找所有 a 标签
soup.find_all('a')
# 查找所有 a 标签和 b 标签
soup.find_all(['a','b'])
  2. 通过正则表达式查找元素
    # 以 b 开头的标签查找
soup.find_all(re.compile("^b"))
  3. 通过属性查找元素
    soup.find_all(
 attrs={
   "属性名":"值"
 }
)
  4. 通过文本内容查找元素
    soup.find_all(text="Elsie")
  5. 通过关键词参数查找元素
    soup.find_all(class="sister")
soup.find_all(id='link2')
  6. 可以混合一起使用
    soup.find_all(
 '标签名',
 attrs={
   "属性名":"值"
 },
 text="内容"
)

select 元素查找

作用

使用 select 函数可以通过 css 样式选择器进行元素查找

css 样式选择器

  • 类选择器 使用 .开头
      soup.select('.clsname')
  • ID选择器 使用 #开头
      soup.select('#id')
  • 标签选择器 使用 标签名
      soup.select('p')
  • 属性选择器 使用 [属性=值]
      soup.select('p[属性="值"]')
  • 层级选择器
    • div p 选择 <div> 元素内部的所有 <p> 元素。
    • div , p 选择所有 <div> 元素和所有 <p> 元素。
    • div > p 选择父元素为 <div> 元素的所有 <p> 元素。
    • .cls1.cls2 选择类名是cls1 并且 类名是cls2

案例一、腾讯招聘

爬取目标

https://hr.tencent.com/

分析

页数参数为 start 从 0 开始，每页增长 10

爬取流程

  1. 先爬取单页数据
  2. 循环页数爬取所有数据

数据提取流程

  1. 先提取<tr> 数据
    1. <tr>的类分别是 class="even" 和 class="odd"
    2. 使用 select 方法提取 <tr> 使用规则 .even,.odd
  2. 在<tr>标签中提取详情数据

实现代码

#!/usr/bin/python3
# -*- coding: utf-8 -*-
import requests
from bs4 import BeautifulSoup
import json
class TencentSpider():

    def __init__(self):
        self.base_url = "https://hr.tencent.com/position.php?lid=&tid=&keywords=&start={}#a"
        self.headers = {
            "User-Agent":"Mozilla/5.0 (Macintosh; Intel Mac OS X 10_14_0) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/71.0.3578.98 Safari/537.36"
        }
        pass

    def get_url_list(self):
        return [self.base_url.format(start) for start in range(0,2821,10)]

    def get_html(self,url):
        response = requests.get(url,headers=self.headers)
        return response.text

    def get_items(self,html):
        items = []
        soup = BeautifulSoup(html,'lxml')
        tr_list = soup.select('.even , .odd')
        for tr in tr_list:
            item = {}
            td_list = tr.find_all('td')
            item["name"] = td_list[0].a.get_text()
            item["cate"] = td_list[1].get_text()
            item["count"] = td_list[2].get_text()
            item["location"] = td_list[3].get_text()
            item["time"] = td_list[4].get_text()
            items.append(item)
        return items

    def save_item(self,item):
        print(item)

    def run(self):

        # 1. 获取 url 列表
        url_list = self.get_url_list()

        all_items = []
        for url in url_list:
            # 2. 发送请求获取html
            html = self.get_html(url)

            # 3. 通过 html 提取数据
            items = self.get_items(html)

            # 4. 保存数据
            for item in items:
                self.save_item(item)
                all_items.append(item)
                # break
            break

        # 把所有items保存到文件中
        with open('09-data.json','w',encoding='utf-8') as f:
            json.dump(all_items,f,ensure_ascii=False,indent=4)

        pass

if __name__ == '__main__':
    spider = TencentSpider()
    spider.run()

    • get("属性名") 获取属性值
    • .children 和 .contents 获取所有子节点
  • beautifulsoup4 中 find_all 和 find 的使用
    • find 返回符合条件的第一个元素
    • find_all 返回符合条件的所有元素
  • beautifulsoup4 中 select 的使用
    • select 可以使用 css 样式选择器对元素进行查询
